# Sneaky
Sneak game in python, the purpose is to make it work using reinforcement learning algorithm


# Use
python3.5+

To make it work you need to install the require libraries.
Run in your terminal

```
pip install -r requirements.txt
```

Then simply run 

```
python3 main.py
```


Enjoy :) !
